# Project Spec: Daily Routine (Productivity Dashboard)

A high-performance, bento-grid style productivity dashboard with real-time performance tracking and AI integration.

## 🎯 Core Objectives
- Provide a clear, actionable view of daily tasks.
- Visualize productivity with 3D isometric charts and trend graphs.
- Integrate AI for task suggestions and personalized insights.
- Maintain a premium "Liquid Glass" dark aesthetic.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (Styling).
- **Backend**: Node.js, Express.
- **Database**: SQLite (local storage).
- **AI**: NVIDIA NVIDIA AI (Qwen 2.5) via backend integration.

## 📐 Key Features
- **3D Progress Pie**: Dynamic rotational isometric chart.
- **Progress Regulation**: Sparkline tracking completion trends.
- **Smart Date Scheduling**: Date-based task management with table preview.
- **AI Suggestions**: BMR-inspired task prioritization and chat support.
- **Heatmap**: Activity tracking over time.
